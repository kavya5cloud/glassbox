# Contributing to Glassbox

Thanks for your interest in contributing!

Glassbox is an open-source developer tool for debugging and understanding LLM applications. Every contribution, whether it's code, documentation, or bug reports, is appreciated.

---

## Getting Started

Clone the repository:

```bash
git clone https://github.com/kavya5cloud/glassbox.git
cd glassbox

'''

---

## One suggestion

> *Glassbox is still in its early days. Whether you're fixing a typo, improving the UI, or adding a new provider, every contribution helps shape the project. Thanks for being part of the journey.*

It gives the project a human touch without feeling overly formal, and it invites people to contribute.
